00030 /**
00031  * A representation of an object stored in remote Object Storage.
00032  *
00033  * A remote object is one whose canonical copy is stored in a remote
00034  * object storage. It represents a local (and possibly partial) copy of
00035  * an object. (Contrast this with HPCloud::Storage::ObjectStorage::Object)
00036  *
00037  * Depending on how the object was constructed, it may or may not have a
00038  * local copy of the entire contents of the file. It may only have the
00039  * object's "metadata" (information such as name, type, modification
00040  * date, and length of the object). Or it may have all of that in
00041  * addition to the entire content of the file.
00042  *
00043  * Remote objects can be modified locally. Simply modifying an object
00044  * will not result in those modifications being stored on the remote
00045  * server. The object must be saved (see

00318   /**
00319    * Get the content of this object as a file stream.
00320    *
00321    * This is useful for large objects. Such objects should not be read
00322    * into memory all at once (as content() does), but should instead be
00323    * made available as an input stream.
00324    *
00325    * PHP offers low-level stream support in the form of PHP stream
00326    * wrappers, and this mechanism is used internally whenever available.
00327    *
00328    * If there is a local copy of the content, the stream will be read
00329    * out of the content as if it were a temp-file backed in-memory
00330    * resource. To ignore the local version, pass in TRUE for the
00331    * $refresh parameter.
00332    *
00333    * If the content is coming from a remote copy, the stream will be
00334    * read directly from the underlying IO stream.
00335    *
00336    * Each time stream() is called, a new stream is created. In most
00337    * cases, this results in a new HTTP transaction (unless $refresh is
00338    * FALSE and the content is already stored locally).
00339    *
00340    * The stream is read-only.
00341    *
00342    * @param boolean $refresh
00343    *   If this is set to TRUE, any existing local modifications will be ignored
00344    *   and the content will be refreshed from the server. Any
00345    *   local changes to the object will be discarded.
00346    * @retval resource
00347    *   A handle to the stream, which is already opened and positioned at
00348    *   the beginning of the stream.
00349    */
00350   public function stream($refresh = FALSE) {
00351 
00352     // If we're working on local content, return that content wrapped in
00353     // a fake IO stream.

00458   /**
00459    * Check whether there are unsaved changes.
00460    *
00461    * An object is marked "dirty" if it has been altered
00462    * locally in such a way that it no longer matches the
00463    * remote version.
00464    *
00465    * The practical definition of dirtiness, for us, is this: An object
00466    * is dirty if and only if (a) it has locally buffered content AND (b)
00467    * the checksum of the local content does not match the checksom of
00468    * the remote content.
00469    *
00470    * Not that minor differences, such as altered character encoding, may
00471    * change the checksum value, and thus (correctly) mark the object as
00472    * dirty.
00473    *
00474    * The RemoteObject implementation does not internally check dirty
00475    * markers. It is left to implementors to ensure that dirty content is
00476    * written to the remote server when desired.
00477    *
00478    * To replace dirty content with a clean copy, see refresh().
00479    */
00480   public function isDirty() {
00481 
00482     // If there is no content, the object can't be dirty.
00483     if (!isset($this->content)) {
00484       return FALSE;
00485     }
00486 
00487     // Content is dirty iff content is set, and it is
00488     // different from the original content. Note that
00489     // we are using the etag from the original headers.
00490     if ($this->etag != md5($this->content)) {
00491       return TRUE;
00492     }
00493 
00494     return FALSE;

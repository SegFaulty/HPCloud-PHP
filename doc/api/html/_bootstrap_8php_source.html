00034  *
00035  * There is no requirement that this class be used. HPCloud is
00036  * built to be flexible, and any individual component can be

00042  *
00043  * - <em>Configuration:</em> "global" settings are set here.
00044  *   See the setConfiguration() method to see how they

00059  * @code
00060  * <?php

1 # Using the HPCloud PHP API
2 
3 This tutorial explains how you can use the PHP API to connect to your HP
4 Cloud services and interact programmatically.
5 
6 ## Object Storage (Swift)
7 
8 One of the services that HP Cloud offers is called "Object Storage".
9 This service provides a useful means of storing objects (usually files)
10 on a service that you control, but that is available to other services
11 in your cloud (and optionally is availably publically).
12 
13 This section of the tutorial describes how you can write PHP code to
14 interact with the Object Storage service.
15 
16 ## Authenticating to Object Storage
17 
18 There are two ways to authenticate to Object Storage:
19 
20 - Legacy Swift authentication
21 - Control Services authentication (Coming soon!)
22 
23 For legacy swift authentication, you will need to use your Account ID
24 and your secret Key, along with the URL to the Object Storage endpoint.
25 If you are an existing HP Cloud customer, you can find all of this
26 information on your console dashboard.

28 ### Using Stream Wrappers
29 
30 There are two main methods for accessing HPCloud through this library.
31 The first is through PHP *stream wrappers*. In PHP, stream wrappers
32 provide a facility with which you can access various data streams (like
33 a webpage, the data in a ZIP file, or an HP Cloud object store) as if
34 they were local files on your file system.
35 
36 Stream wrappers have a huge advantage for you: You can use the normal
37 file system functions (`fread()`, `mkdir()`, `file_get_contents()`, etc)
38 to access things not necessarily on your local filesystem. The HP Cloud
39 library integrates with this facility of PHP.

42 ### Using the HPCloud Classes
43 
44 While the stream wrappers are a fantastic way to accomplish many common
45 tasks, sometimes you need a finer level of control, or you wish to use
46 an Object-Oriented API. We provide you with the classes you need to work
47 this way.
48 
49 (Deep dark secret: Actually, these are the classes that underly the
50 stream wrappers.)

52 In this section of the tutorial, we focus on using this API as a
53 data-access layer.

62 ## Slightly Irreverant Glossary
63 
64 *Account ID:* You service provider (ahem, HPCloud) will provide you with
65 an account ID and a secret key, along with a URL, that can be used to
66 access the cloud APIs.
67 
68 *Container:* A namespace extension useful for differentiating object
69 space with pseudo-containment logical units. Or, a directory. (see
70 _Object Storage_)
71 
72 *Object Storage:* A service provided by OpenStack (and implemented by
73 HPCloud) that allows you to store entire files on the cloud. Files can
74 be organized into _containers_, which are rough analogs to file system
75 directories (or folders).

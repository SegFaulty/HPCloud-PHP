00001 # Using the HPCloud PHP API
00002 
00003 This tutorial explains how you can use the PHP API to connect to your HP
00004 Cloud services and interact programmatically.
00005 
00006 ## Object Storage (Swift)
00007 
00008 One of the services that HP Cloud offers is called "Object Storage".
00009 This service provides a useful means of storing objects (usually files)
00010 on a service that you control, but that is available to other services
00011 in your cloud (and optionally is availably publically).
00012 
00013 This section of the tutorial describes how you can write PHP code to
00014 interact with the Object Storage service.
00015 
00016 ## Authenticating to Object Storage
00017 
00018 There are two ways to authenticate to Object Storage:
00019 
00020 - Legacy Swift authentication
00021 - Control Services authentication (Coming soon!)
00022 
00023 For legacy swift authentication, you will need to use your Account ID
00024 and your secret Key, along with the URL to the Object Storage endpoint.
00025 If you are an existing HP Cloud customer, you can find all of this
00026 information on your management dashboard.

00028 ### Using Stream Wrappers
00029 
00030 There are two main methods for accessing HPCloud through this library.
00031 The first is through PHP *stream wrappers*. In PHP, stream wrappers
00032 provide a facility with which you can access various data streams (like
00033 a webpage, the data in a ZIP file, or an HP Cloud object store) as if
00034 they were local files on your file system.
00035 
00036 Stream wrappers have a huge advantage for you: You can use the normal
00037 file system functions (`fread()`, `mkdir()`, `file_get_contents()`, etc)
00038 to access things not necessarily on your local filesystem. The HP Cloud
00039 library integrates with this facility of PHP.
00040

00042 ### Using the HPCloud Classes
00043 
00044 While the stream wrappers are a fantastic way to accomplish many common
00045 tasks, sometimes you need a finer level of control, or you wish to use
00046 an Object-Oriented API. We provide you with the classes you need to work
00047 this way.

00049 (Deep dark secret: Actually, these are the classes that underly the

00062 ## Slightly Irreverant Glossary
00063 
00064 *Account ID:* You service provider (ahem, HPCloud) will provide you with
00065 an account ID and a secret key, along with a URL, that can be used to
00066 access the cloud APIs.
00067 
00068 *Container:* A namespace extension useful for differentiating object
00069 space with pseudo-containment logical units. Or, a directory. (see
00070 _Object Storage_)
00071 
00072 *Object Storage:* A service provided by OpenStack (and implemented by
00073 HPCloud) that allows you to store entire files on the cloud. Files can
00074 be organized into _containers_, which are rough analogs to file system
00075 directories (or folders).
